Dynaform.js

Dynaform form made easy.

  • Version 1.0
  • ·
  • Download
  • ·
  • Github project

Dynaform.js is a small JavaScript library which allows you to create programmatic form boxes using jQuery, without having to worry about creating, managing or removing any of the required DOM elements or JS event handlers.

Download Version 1.0

The latest stable version of Bootbox is 1.0. You can download the minified library dynaform.min.js for use in production, or the uncompressed dynaform.js for use in development. I wouldn’t recommend hotlinking to either file as Github does not serve the correct mime type which may cause security warnings to be triggered based on your browser’s settings.
